# Neuro-symbolic logical problem solver
Коллоквиум по МЛиТА

**Презентация:** https://docs.google.com/presentation/d/1MMdkUs3JBgxxBAaeyo_ZuPCqhSgk3VHlLQDeuffCM00/edit?usp=sharing

**Состав команды:** Копасова Ксения 3384, Мокрушина Вероника 3384, Самойлова Екатерина 3384

**Указания для использования приложения:**
- установка Ollama с официального сайта https://ollama.com/
- после установки Ollama необходимо загрузить модель DeepSeek: ollama pull deepseek-v3.1:671b-cloud
- установка файла gui_app.exe из репозитория проекта
- запуск приложение осуществляется двойным щелчком мыши по файлу gui_app.exe

## Описание главных файлов проекта
В папке src находятся основные файлы проекта.

***gui_app.py***: графический интерфейс системы логического доказательства методом резолюций.
  
  *Основные функции*:
  - Ввод логических высказываний на естественном языке
  - Автоматическая формализация в логические формулы
  - Визуализация процесса доказательства
  - Генерация объяснений на простом языке
  - Готовые примеры для тестирования
    
  *Интерфейс*:
  - Поле ввода - для текстовых высказываний
  - Вкладки:
    - Формализованные формулы
    - Детали доказательства
    - Объяснение
  - Панель управления - кнопки доказательства и очистки
  - Примеры - готовые высказывания для быстрого тестирования

***logic_formalizer.py***: модуль для преобразования текстовых задач в формальные логические выражения с использованием языковой модели Ollama.

  *Основные функции*:
  - Автоматическая формализация естественно-языковых описаний
  - Преобразование в логику предикатов первого порядка
  - Подготовка формул для метода резолюций

  *Алгоритм преобразований*:
  1. Предваренная нормальная форма (ПНФ) - вынос кванторов
  2. Сколемовская нормальная форма (СНФ) - устранение кванторов существования
  3. Удаление кванторов всеобщности
  4. Конъюнктивная нормальная форма (КНФ)
  5. Разбиение на дизъюнкты

***resolution_engine.py***: движок для автоматического доказательства теорем методом резолюций в логике предикатов первого порядка.

  *Основные функции*:
  - Автоматическое доказательство логических утверждений
  - Поиск противоречия через вывод пустой клаузы
  - Подробное логирование всех шагов доказательства

  *Алгоритм преобразований*:
  1. Унификация - сопоставление термов с подстановками
  2. Резолюция - устранение комплементарных литералов
  3. Проверка тавтологий - исключение избыточных клауз
  4. Поглощение - удаление подмножеств существующих клауз

***proof_explainer.py***: модуль для преобразования формальных логических доказательств в понятные объяснения на естественном языке с использованием языковой модели Ollama.

  *Основные функции*:
  - Генерация человеко-читаемых объяснений шагов доказательства
  - Преобразование формальной логики в понятный русский язык
  - Создание связных текстовых объяснений

### Тестирование системы
Чтобы убедиться, что все части программы работают правильно, мы создали тесты для каждого модуля:

***test_formalizer.py*** - проверяет перевод текста в логические формулы

***test_resolution.py*** - тестирует алгоритм доказательства

***test_explainer.py*** - проверяет создание понятных объяснений
