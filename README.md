# Neuro-symbolic logical problem solver
Коллоквиум по МЛиТА

Презентация: https://docs.google.com/presentation/d/1MMdkUs3JBgxxBAaeyo_ZuPCqhSgk3VHlLQDeuffCM00/edit?usp=sharing

Состав команды: Копасова Ксения 3384, Мокрушина Вероника 3384, Самойлова Екатерина 3384

Указания для использования приложения:
- установка Ollama с официального сайта https://ollama.com/
- после установки Ollama необходимо загрузить модель DeepSeek: ollama pull deepseek-v3.1:671b-cloud
- установка файла gui_app.exe из репозитория проекта
- запуск приложение осуществляется двойным щелчком мыши по файлу gui_app.exe
